# Genesis
This is my submission for the Very Serious Junper Dev Game Jam. I ended up settling on the name 'Genesis' due to the headcanon lore I have about this game.
## Theme
The theme this year is "spin to win".
## Inspiration
I'm mainly taking inspiration from older games, specifically Atari games that utilized the paddle. Tempest, Arkanoid / Breakout, and Pong were big inspirations for figuring out what I wanted to do in terms of gameplay.
